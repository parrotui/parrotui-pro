##  安装 / Installation

```bash
# 开源版 (免费)
npm install parrotui

# 商业版 (需要授权密钥)
npm install parrotui-pro
```

### 配置授权密钥 / Configure License Key

```javascript
// 初始化 ParrotUI Pro
import { ParrotUI } from 'parrotui-pro';

ParrotUI.init({
  licenseKey: 'YOUR_LICENSE_KEY' // 购买后获取
});
```
